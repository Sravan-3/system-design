# 🚀 System Design Playground

A hands-on repository focused on **learning system design by building real systems**.

Instead of only studying theory and diagrams, this project explores how distributed systems behave in practice — under load, across services, and with real infrastructure components.

---

## 🧠 What This Is About

* Turning **system design concepts into working implementations**
* Understanding **scalability, performance, and bottlenecks**
* Experimenting with **real-world architectures**
* Learning by **breaking, testing, and improving systems**

---

## ⚙️ Tech Stack

* Java, Python
* Docker & Docker Compose
* PostgreSQL, MongoDB
* Redis
* NGINX
* k6 (load testing)
* Terraform

---

## 🚀 Getting Started

```bash
git clone https://github.com/Sravan-3/system-design.git
cd system-design
```

Run any module independently using Docker:

```bash
docker-compose up
```

---

## 🧪 What You Can Do

* Simulate high traffic and observe system behavior
* Experiment with scaling services horizontally
* Test performance with and without caching
* Explore different communication patterns
* Identify bottlenecks and optimize systems

* System Design Interview Preparation
